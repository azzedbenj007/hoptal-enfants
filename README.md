# Hôpital des Enfants Al-Amal

Site web officiel de l'**Hôpital des Enfants Al-Amal** — plateforme d'excellence pédiatrique au Maroc.

## Aperçu

Site vitrine responsive présentant les services, l'équipe médicale et les informations pratiques de l'hôpital. Conçu pour offrir une expérience chaleureuse et accessible aux familles.

## Fonctionnalités

- **Accueil** — Hero section avec présentation et appel à l'action
- **Spécialités** — Cartes des services pédiatriques disponibles
- **Innovation** — Technologies et équipements médicaux modernes
- **Mot de la Directrice** — Message institutionnel de la direction
- **Équipe médicale** — Présentation des médecins
- **Prise de rendez-vous** — Formulaire de contact en ligne
- Navigation responsive (mobile + desktop)

## Stack technique

| Technologie | Usage |
|---|---|
| HTML5 | Structure |
| [Tailwind CSS CDN](https://tailwindcss.com) | Styles utilitaires |
| [Font Awesome 6](https://fontawesome.com) | Icônes |
| Google Fonts (Nunito, Poppins) | Typographie |

## Lancer le projet

```bash
# Cloner le dépôt
git clone https://github.com/azzedbenj007/hoptal-enfants.git
cd hoptal-enfants

# Ouvrir directement dans le navigateur
open index.html
```

Aucune dépendance à installer — le projet fonctionne directement en ouvrant `index.html` dans un navigateur.
